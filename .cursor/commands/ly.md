# /ly

List all available ly-* commands.

## Usage

Type `/ly` in Cursor chat to see all available ly-* commands.

## What it does

Displays a list of all available ly-* commands with their descriptions and usage.

## Example

```
/ly
```

## Execution

When this command is invoked, execute the following script:

```bash
bash scripts/ly.sh
```

The script is located at `${workspaceFolder}/scripts/ly.sh` and must be executed from the workspace root.
